Basic SQL Commands

Master these fundamental SQL commands to start querying databases effectively. Each command plays a crucial role in retrieving and organizing your data.

SELECT

The SELECT statement is used to retrieve data from a database. You specify which columns you want to see in your results. 

SELECT column1, column2
FROM table_name;

Example:

SELECT first_name, last_name, email
FROM customers;

This retrieves the first name, last name, and email of all customers.

FROM

The FROM clause specifies which table you want to retrieve data from. It's required in every SELECT statement. 

SELECT *
FROM table_name;

Example:

SELECT *
FROM products;

The asterisk (*) means "all columns" - this retrieves every column from the products table.

WHERE

The WHERE clause filters records based on specific conditions. Only rows that meet the condition are included in the results. 

SELECT column1, column2
FROM table_name
WHERE condition;

Example:

SELECT product_name, price
FROM products
WHERE price > 50;

This retrieves only products with a price greater than $50.

ORDER BY

The ORDER BY clause sorts your results by one or more columns. You can sort in ascending (ASC) or descending (DESC) order. 

SELECT column1, column2
FROM table_name
ORDER BY column1 ASC;

Example:

SELECT product_name, price
FROM products
ORDER BY price DESC;

This retrieves all products sorted by price from highest to lowest.

Putting It All Together

Here's a complete query that uses all four commands together: 

SELECT product_name, category, price
FROM products
WHERE category = 'Electronics'
  AND price < 1000
ORDER BY price DESC;

This query retrieves the name, category, and price of all electronics products under $1000, sorted from most to least expensive.
